Task01:
1. В исходном коде глобально используется «using namespace std;», это загрязняет глобальное пространство имён. Для всего кода заменено на префекс «std::».
2. Присутствует некорректное название функции «findAnswer». Данное название не соответствует стилю именования (PascalCase). Функция переименована в «FindAnswer».
3. В функции «FindAnswer» некорректно расположены скобки (открывающая скобка находиться на той же строке).Исправлено - открывающая скобка находится на новой строке. Также данная проблема исправлена не только в данной функции, но и во всей программе.

5. Были заменены некорректные имена переменных во всей программе. Они стали соответствовать формату PascalCase: первая буква каждого слова в названии пишется с заглавной буквы.
6. Переименованы все названия векторов по тому же формату PascalCase.
7. Использовалась не стандартное имя переменной отвечающей за цикл «for». Переменная «s» заменена на привычную переменную «i».

Task04:
1. В исходном коде глобально используется «using namespace std;», это загрязняет глобальное пространство имён. Для всего кода заменено на префикс «std::».
2. Некорректное расположение скобок (открывающая скобка находится на той же строке). Исправлено — открывающая скобка находится на новой строке (стиль Allman) во всех функциях, циклах и условных операторах.
3. Присутствуют некорректные названия функций, не соответствующие стилю именования PascalCase и не отражающие суть выполняемых операций. Функции переименованы: старые имена названые «recnum», «delen», «primer» были заменены на «ParseNumber», «ParseMultiplicationAndDivision», «ParseAdditionAndSubtraction» соотвецтвенно.
4. Присутствуют некорректные имена переменных, не соответствующие стилю именования PascalCase, использующие транслит, слишком короткие и неинформативные. Переменные переименованы в нове имена удовлетворяющие всем критериям.

8. В функции «ParseMultiplicationAndDivision» и «ParseAdditionAndSubtraction» отсутствовали скобки при условных операторах «if». Добавлены фигурные скобки для всех ветвлений.
